# File Tools

## What is File Tools?

File Tools is a Python library to manage your files easily.  
It aims to simplify common file operations in Python, making it useful for application development and everyday scripting.

## Example usage

```python
from filetools import backup

file = "C:/Users/example.txt"
backup_location = "C:/Users/backups"
created_backup = create_backup(file=file, where=backup_location)
```

This is how it is expected to be.
